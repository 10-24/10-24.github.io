osu!(lazer) Updates: April 13, 2024
#

警告：看到这条新闻的第一眼你就已经被Laser包围了，这可能导致超级棒的Lazer更新增多！

新的一月带来了又一轮的更新，您可以通过阅读下文或观看视频摘要来了解。

较小的变化
#

  • 现在在 osu!taik，osu!catch 和 osu!mania 中暂停后有倒计时。给你一点准备时间，而不是太快地把你扔进战斗。
  • 在 osu!catch 的香蕉雨中不会失败了。catch社区终于可以安心享受香蕉雨了。
  • 旋转工具按钮现在即使只选择了 1 个打击圈也能工作。对于制图社区来说，这是一个小而受欢迎的改变。
  • 支持在歌曲选择中通过多个谱面状态进行搜索这。意味着你可以输入 status=loved,ranked 来获取 Loved 和 Ranked 的地图。查看此 wiki文章以获取所有搜索过滤器的列表。
  • 编辑器性能改善。感谢 EVAST!

各种视觉变化
#

现在，未命中的击打会显示在击打精度热图上。这意味着你终于可以非常方便地确认游戏区域的左上角是否真的无法击中。

与地图状态（如 Loved 或 Qualified）相关的免责声明已从 osu!(stable)中移植过来。它们还经过了快速的设计修改，以便与癫痫警告或未来可能的其他类型的通知同时显示。

击打误差表上新增了图标，兔子和乌龟分别代替了卡车和自行车，表示击打是早还是晚。

如果你的游戏是在一个未排名的地图或者 mod 组合上进行的，结果屏幕上的pp数值会被灰色显示。然而，对于那些喜欢在他们的 Reddit 得分帖子上写上"如果有pp的话"的人来说，这些值仍然是可见的，也是准确的。

出于某种奇怪的原因，pippidon 在 osu!(lazer)中被缩小了，所以开发者决定结束这种情况，恢复她以前的尺寸。

Catmull 滑条（出现在非常旧的地图上）没有被正确地渲染。这种差异导致了一种现象，一些滑条轻微地显示出"突起"或者，也许在制图社区更常见的称呼–“肿瘤”。我们很高兴地说，catmull 滑条现在已经没有癌症了。

主菜单轮换横幅
#

主菜单不时会显示正在进行的活动的图形。已经增加了支持（在两个客户端中）以在多个活动同时进行的情况下进行切换

新的实验性渲染器
#

游戏中添加了一些新的实验性“延迟”渲染器：

如果你遇到了图形或性能问题（甚至如果你没有），试试新的渲染器，并查看这个 this GitHub thread，你会发现一些问题以确定它们是否解决了任何问题。

你可能不应该期待任何FPS的提升，但是新的渲染器对于一些人来说可能会更加可靠。

如果你喜欢技术细节， smoogipoo（实施这一变更的开发者）已经写了一篇优秀的 博客文章，覆盖了这个过程中的所有事情以及它是如何运作的。

osu!(lazer)的下一步
#

在 osu!(lazer)的pp上线后…开发者们一直忙于处理后续事务。然而，除此之外，未来基本上还未确定。

实际上，这是开发者决定让社区决定接下来要关注什么的地方。

这可能是任何一个能够在osu!社区内提高参与度的伟大想法。开发者已经列出了一份 未来任务清单，但在将这些任务投票决定之前，我们的目标是看看是否有遗漏的东西。社区中的人们是否有着强烈的热情希望看到的东西。

无论是更具创新性和功能性的编辑器，pp用于调整模式的速率，对等级系统的全面改革，Elo/MMR匹配系统，还是完全不同的其他内容，这都取决于你。

这是计划：

  1. 在此更新的 YouTube 视频评论中提供想法。如果你喜欢别人的想法，只需对其进行投票。
  2. 开发者将采纳最受欢迎的内容，并通过社区投票来决定 osu!(lazer)的下一步行动。
